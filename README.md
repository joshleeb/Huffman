# Huffman

Simple Huffman Encoder/Decoder binary.

## Motivation

Wanted to work on a small side project to get better at programming in C++.

## Usage

```bash
huffman [options]
```

Input is read from `stdin` and outputted to `stdout`.

### Options

#### Encoding

`-e --encode`

Encode the input.

#### Decoding

`-d --decode`

Decode the input.
